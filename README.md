# Flatris (Node.js)

A simple Flatris (Tetris-style) game built with Node.js and vanilla JavaScript in the browser.

## Project Structure

```text
flatris/
├─ package.json
├─ server.js
├─ README.md
└─ public/
   ├─ index.html
   ├─ styles.css
   └─ app.js
```

## Prerequisites

- Node.js 18+ (Node.js 20 recommended)

## Run Locally

1. Start the server:

```bash
npm start
```

2. Open in your browser:

```text
http://localhost:3000
```

## Controls

- Left / Right arrow: Move block
- Down arrow: Soft drop
- Up arrow: Rotate block
- Space: Hard drop
- R: Restart game
